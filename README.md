# AgriSmart AI

An AI-powered Smart Agriculture Assistant developed using:

- OutSystems
- Python Flask
- Google Gemini API
- OpenWeather API

## Features

- Farmer Registration
- Weather Dashboard
- Crop Recommendation
- Soil Analysis
- Fertilizer Recommendation
- Crop Calendar

## Technologies

- OutSystems
- Python
- Flask
- Google Gemini
- OpenWeather API
- Render
